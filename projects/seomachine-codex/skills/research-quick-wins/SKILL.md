# Research Quick Wins

Run the quick-win keyword analysis.

## Command

```bash
.venv/bin/python research_quick_wins.py
```

## Output

The script writes a report to `research/quick-wins-<date>.md`.

## Best Follow-Up

- Existing URL present -> `analyze-existing` then `optimize`
- No useful URL -> `research` then `write`
