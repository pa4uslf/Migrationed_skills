# Research Trending

Run the trending-query analysis.

## Command

```bash
.venv/bin/python research_trending.py
```

## Output

The script writes a report to `research/trending-<date>.md`.

## Best Follow-Up

- Use `research-serp` on critical trends
- Use `write` or `article` immediately for trends with strong fit
